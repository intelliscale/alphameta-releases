# AlphaMeta Releases

Public binary releases for AlphaMeta — an agentic trading tool for Interactive Brokers.

## Download

Each [release](https://github.com/intelliscale/alphameta-releases/releases) includes:

- \`alphameta-{version}-macos-arm64.tar.gz\` — compiled binary for macOS ARM64
- \`checksums.txt\` — SHA-256 checksums

## Installation

```bash
curl -fsSL https://alphameta.app/install.sh | bash
```

## Start

```bash
alphameta start
```

## Agent Skills

After starting the server, use AlphaMeta via agent skills: [`alphameta-skills`](https://github.com/intelliscale/alphameta-skills)
